# Object detection models (AutoModelForObjectDetection)

These models are compatible with the Hugging Face Vision filter (`AutoImageProcessor` + `AutoModelForObjectDetection`).

| MODEL_ID | REVISION |
|----------|----------|
| facebook/detr-resnet-50 | main |
| microsoft/conditional-detr-resnet-50 | main |
| PekingU/rtdetr_r50vd | main |
| PekingU/rtdetr_r18vd | main |
| PekingU/rtdetr_r101vd | main |

## Example .env

```bash
MODEL_ID=PekingU/rtdetr_r50vd
REVISION=main
```

Replace `MODEL_ID` and `REVISION` with any row from the table above.
